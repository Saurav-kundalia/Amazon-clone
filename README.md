# Amazon-clone
A beginner-friendly Amazon homepage clone built with HTML and CSS, featuring responsive design, navigation, and product sections. Demonstrates foundational web development skills.

## Features
- Responsive layout
- Navigation with search bar
- Product boxes with images
- Footer with links

## Technologies Used
- HTML5
- CSS3
- Font Awesome (for icons)

## How to Run
1. Clone the repo: `git clone [your-repo-url]`
2. Open `index.html` in a browser.
3. Ensure images (e.g., amazon_logo.png) are in the same folder.
